#. To configure the default behaviour for automatic package creation on
   shipping method creation:

    * Go to Inventory > Configuration > Settings > Shipping
    * Check the 'Automatic Package Creation Default' box

#. To change a particular shipping method:

    * Go to Inventory > Configuration > Delivery > Shipping Methods
    * Choose the one you want to modify and check or uncheck the
      'Automatic Package Creation' box.
