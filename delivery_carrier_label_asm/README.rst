==========================
Delivery Carrier Label ASM
==========================

This module adds ASM integration in version 12.0:
- Get labels from ASM in diferents formats.
-


Usage
=====

Set the ASM config parameter with your ASM username to do the connection with
the API.

In picking user ASM delivery carrier and choose the label type.
