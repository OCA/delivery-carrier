This module allows to set a domain of applicability on products when calculating shipping charged based on rules.
A possible use case could be: when calculating shipping fee using a rule based on price of products, only take into consideration price of storable and consumable products.

**Table of contents**

.. contents::
   :local:

Usage
=====

Go to Sales > Configuration > Shipping methods, select or create a record with Provider = Based on rules
Open a pricing rule > field "Apply to products" is available to set which products should be taken into consideration in both "Condition" and "Delivery cost" fields.
