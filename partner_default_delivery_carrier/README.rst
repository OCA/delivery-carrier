Allows defining a default delivery carrier for new partners.

The default value will be set in the Sales configuration, and will be
company-specific.
Whenever a new partner is created, field "Delivery Method" will be filled with
the company's default carrier.

.. contents::

Configuration
=============

In "Sales > Configuration > Settings > Shipping" an option will be available
to choose the default delivery carrier (by company). It will be used to prefill
field "Delivery Method" on new partners.
